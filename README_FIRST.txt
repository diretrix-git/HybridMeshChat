  🚀 BUILD THE APP (ONE COMMAND)
================================================================================

1. Connect your Android phone via USB
2. Enable USB Debugging (Settings → Developer Options)
3. Run this command:

   npx expo run:android

4. Wait 3-5 minutes
5. App installs and launches automatically!

================================================================================
  📱 TEST HOTSPOT MODE
================================================================================

Host Phone (Person 1):
  1. Create WiFi hotspot (Settings → Hotspot)
  2. Start server: cd server && npm install && npm start
  3. Open app, enter: 192.168.43.1:3000, room: friends-chat, password: secret123

Guest Phones (Person 2, 3, 4...):
  1. Connect to host's WiFi hotspot
  2. Open app, enter SAME details as host
  3. Start chatting!

No internet needed! All messages stay local on the hotspot network.
